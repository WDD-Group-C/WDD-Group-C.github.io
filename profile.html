Using the Profile Creator

  1. Click the START button to begin the registration process.
  2. Use the Next button to submit your response and proceed to the next question.
  3. Click the Skip button to skip the current question.
  4. Click on the box above the Next and Skip buttons to select the prompt box and enter text.

Keyboard Controls

  1. P - Focuses on the prompt box for entering text.
  2. N - Submits the current response and moves to the next question.
  3. S - Skips the current question.
  4. Enter - Starts the creator or to finish submitting a skipped prompt.
  5. Use Tab to navigate through "Skipped" prompts, then use ↑ and ↓ arrows to move between them.

Mouse controls are also available and are fully functional
